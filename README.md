Covid-19 Data Exploration | SQL Project

This project explores global Covid-19 data using SQL Server. It analyzes death rates, infection rates, and vaccination rollouts across countries and continents using real-world data sourced from Our World in Data.
Skills Demonstrated:

Joins
CTEs (Common Table Expressions)
Temp Tables
Window Functions
Aggregate Functions
Creating Views
Converting Data Types
NULLIF for safe division

Key Questions Explored:

What percentage of the population died after contracting Covid?
What percentage of the population was infected per country?
Which countries had the highest infection and death rates?
Which continents had the highest death counts?
What percentage of each country's population got vaccinated over time?

Tools Used:

Microsoft SQL Server
SQL Server Management Studio (SSMS)

Data Source:
Our World in Data — https://ourworldindata.org/covid-deaths
